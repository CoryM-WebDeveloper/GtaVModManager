1. Click New repository
2. Name it <Program Name Here no spaces>

🛡️ Extra safety (highly recommended)

You can also protect main on GitHub:

Go to:

- GitHub repo → Settings → Branches
- Add branch protection rule for main
    - Require pull requests
    - Block direct pushes (optional but powerful)

This turns main into a sealed “release vault.”


Very Important Development Process Notes:

Switches and creates a branch called dev
git checkout -b dev

Switches to the dev branch
git checkout dev

-u = Tracking parameter. This push to github to the location of the dev branch 
and tracks it. This needs to be done for each branch as it will creates what is
requested and then push it to github.
git push -u origin dev
